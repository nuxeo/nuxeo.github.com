Nuxeo Platform Release Notes

Nuxeo Platform 5.7.2 is now available for download. This is the second Fast Track version, using the newly launched versioning system, with Fast Track on odd-numbered releases and Long Term Support on even-numbered releases.

Below you will find the list of new features and improvements included in this new release:

  • For Users
    • DAM
    • Workflow
    • Video
    • Ergonomy and UX
    • Translation
  • For Administrators
    • Monitoring
    • Multi-Tabs Navigation
    • Nuxeo.conf New Parameters
  • For Developers
    • Content Automation
    • Document Oriented REST API
    • Richfaces
    • CMIS
    • Content Views
    • CORS Support
    • Scan Importer
    • New Operations
  • Upgrade Notes
  • Known Issues
