8.1 Release Notes

For Users

Instant Share

A new feature allows to invite users to a given folder of Nuxeo Platform with either Read or Edit permission, by email. Invited users do not have an account on the Nuxeo Platform instance. The access is temporary and any access remains tracked in the audit log, based on the email adress used.

Live Connect For Box

An addon is available for using Live Connect with Box.com. Live Connect allows to easily reference files that are stored in the cloud and have them participate to your business processes.

User Feedback On Permission Management Tab

Feedback using paper-toast has been implemented on permission tabs on create, update and delete permission as well as on the "send notification email" actions.

Smart Search has Been re-aligned

Smart Search, that allows users to build their query has been re-integrated to search tab, consistently with other search options. Smart Search is delivered as a Nuxeo Package.

Confirmation Popup Before Purge

In the Admin Center, when removing permissions of a given user, you now have to confirm on a popup that you really want to purge all the permissions.

"Http Transport returned a 0 status code" Error Has Been Made More Friendly

When this error happens, message is now displayed as a warning and translated in a more friendly way.
