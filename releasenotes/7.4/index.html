Nuxeo Platform 7.4 Release Notes

Nuxeo Platform 7.4 is now available for download. This is the third Fast Track for the next LTS release, using the versioning system.

Below you will find the list of new features and improvements included in this new release:

  • For Users
    • PDF Viewer
    • File Dispatching
    • Media Content Publishing
    • Cloud Services management screen in Home and Admin Center.
    • Web Authentication for Drive
    • Logging of File Download
    • Saved Search
    • Full path is visible on the the root selector of publishing tab
    • Allow to set lifecycle state when using csv import
    • Video Player can be used with any property
  • For Administrators
    • Changing Tomcat Admin Host
    • VCS Full Text Search Enablement and Configuration
    • Elasticsearch Server Version Recommended is Now Elasticsearch 1.5.2
    • Reindexing Optimization and Reindexing on Startup
    • Audit is Stored in Elasticsearch
  • For Developers
    • Automation: Class Filter on Automation Scripting
    • Automation: Platform functions available in Automation Scripts
    • Automation: Extension point for helpers
    • Automation: HTTP Helper
    • Elasticsearch API Passthrough
    • Elasticsearch Hints
    • File Support on Workflow API
    • Reassign and Delegate Tasks Endpoint
    • Get Rendition Operation
    • Conversion
    • Workmanager Find Method
    • JSON Export of Layout and Widgets
    • Pluggable File Widget
    • JavaScript List Widget
    • REST endpoint for CSS and JS Resources
    • oAuth Service Providers Extension Point
    • Web Components and Polymer
    • Removal / Deprecation
  • Upgrade Notes
  • Known Issues
