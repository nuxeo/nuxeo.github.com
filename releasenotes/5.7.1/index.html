Nuxeo Platform Release Notes

Nuxeo Platform 5.7.1 is now available for download. This is the first Fast Track version, using the newly launched versioning system, with Fast Track on odd-numbered releases and Long Term Support on even-numbered releases.

Below you will find the list of new features and improvements included in this new release:

  • For Users
    • Nuxeo Drive
    • DAM
    • Mobile
    • Importer CSV
    • User Experience
    • Search
    • Quota
    • Birt
    • Content Routing
    • Jenkins Report Module
  • For Administrators
    • Right Audit
    • Kerberos Login Module
    • Monitoring
    • Nuxeo.conf new parameters
    • MarketPlace/Update Center
  • For Developers
    • Content Automation
    • New operations
    • CSS 3 Support in Theme Engine
    • UI Style Guide
    • UI Framework
    • Nuxeo Visible Content Store (VCS)
  • Upgrade Notes
  • Known Issues
