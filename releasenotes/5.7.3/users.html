UI/UX Improvements

Select2 Integration

We have redesinged the legacy error page to have something cleaner, with a user understandable error message.

SessionTimeOut feedback

Users see a 'session expired" message when their session expires.

Form submit Shield

This will protect the heavy handed crazy clicker to have errors because they have clicked on the same button many times in a short time span. The user will receive a message tellling to calm down. It's better for his mouse and the server.

Tag to version

When a version of a document is created, the tags on the master document are copied to the new version.

Redesigned Error Page

We have redesinged the legacy error page to have something cleaner, with a user understandable error message.

Workflow

New Workflow Tab

We have a new workflow tab that lets the user see all his assigned tasks at once and filter them.

Task Assignation

Workflow tasks can now be reassigned to someone else.

MultiTask Workflow Node

We have introduced a new option that creates a different task for each assignee of the node.
