Nuxeo Platform 6.0 Release Notes

Nuxeo Platform 6.0 is now available for download. This is a LTS version, using the Nuxeo versioning system, with Fast Track on odd-numbered releases and Long Term Support on even-numbered releases.

Below you will find the list of new features and improvements included in this new release, since the previous LTS (5.8). If you are interested in changes since the previous Fast Track (5.9.5), you should go to the 5.9.5 --> 6.0 release notes.

  • For Users
    • Nuxeo Drive
    • Collections
    • Search Experience - Rebooted
    • Spreadsheet Edit
    • Bulk Creation
    • Nuxeo Drive iOS Edition
    • Full Screen Mode
    • User Invitation
    • UI/UX
    • Favorites
    • Insert a Video in a Note
    • Photoshop Mime-types Correctly Detected
    • Updated Picture Generation Resolution
    • Simplified ACLs
    • Suggestion Widgets
  • For Administrators
    • User and Groups Providers and Authentication Mode Configuration Wizard
    • AES Binary Store Encryption Mode
    • Private Marketplace Channel
    • Store Password Hash in LDAP User Directory
    • Packages Directory Out of the Data Directory
    • Logging Slow Queries
    • Java version is checked at server startup
  • For Developers
    • Elasticsearch in the Platform
    • MongoDB Backend
    • REST API
    • iOS SDK
    • JavaScript SDK
    • Box, Mule, CMIS: great new integration capabilities
    • Support of Facets and Other Kind of Aggregates on Page Providers and Content Views
    • JSF2, New User Actions Placeholders, Result Layouts and Other Evolutions of the UI Framework and CAP Web Application
    • Changes in Features Layout, Migration to Maven 3
    • Content Automation
    • Authentication and Other User Management Evolutions
    • Audit Service Based on Elasticsearch
    • Various
  • Upgrade Notes
  • Known Issues
