8.10 Release Notes

Farewell

Deprecated Addons

8.10The following addons are deprecated with the release of Nuxeo Platform LTS 2016:

  • Nuxeo Review Workflows Dashboards
  • Smart Search
  • Nuxeo Virtual Navigation
  • Nuxeo Forum
  • Nuxeo Categorization
  • Nuxeo Platform Sync
  • Nuxeo Web Embeded
  • Nuxeo Additional Themes
  • Nuxeo Web Mobile
  • Nuxeo Share Bookmarks / Nuxeo Classification
  • Nuxeo Semantic Entities
  • Box API

End of Life Features

  • 8.2Annotations have been removed.

  • 8.10Everything related to AssetLibrary (coming from legacy DAM application) has also been removed. See upgade documentation for migrating your data.

  • 8.10nuxeo-webengine-features has been removed, removing WebEngine Admin site.
