    • doneNuxeo Web UI Preview Edition

      The Nuxeo Web UI has been rewritten entirely, switching from JSF stack to Web Component (HTML) / Polymer framework, with a completely re-designed interface. It allows enhanced productivity with direct access to last visited documents, collections, easy browsing between prev/next document, full screen suggester. Effort has been made for making the tasks validation flow highly efficient. With all those improvements, Nuxeo Web UI is a first class choice for projects where visual and workflows are important requirements.

    • doneJSF Webapp Extraction

      The JSF Webapp is extracted from Nuxeo Platform distribution, as a Nuxeo Package available for install. This allows to have a lighter default distribution that starts more quickly.

      Provide the ability to manage several branches of the configuration made in a Studio project.

    • doneUnified JavaScript Client

      JavaScript client has evolved in one code base for browsers and node.js and is compatible with React Native. It is now written in ES6 and is compatible with latest API improvements of the platform, including new file upload api. This version of the client also includes new helpers for facilitating the use of the platform: permissions management, conversions, etc ..

      APIs

    • doneNuxeo Java Client: Android/Java Convergence

      The Nuxeo Java Client is refactored for covering both automation resources endpoints, with coverage of the most recent features of our rest api (marshallers, content enrichers, ...) and this client is designed for being used both in plain jav or JEE applications as well as on Android OS.

  • 8.1Fast TrackJanuary 2016April 2016
    • doneLive Connect for Box

      It is possible to invite a user to a folder or a document just with an email address. It will produce a unique link that includes a token that gives either read or write access to the folder, for a given period of time.

  • Search: Synonyms Managementdone

    A synonyms management module, that makes use of Elasticsearch thesaurus capabilities behind the scene, allows administrators of Nuxeo Platform to edit business synonyms for any term. For instance "painkiller" can be synonym of "aspirin". All documents containing painkiller will be returned if user searches for "aspirin".

  • Elasticsearch 2.Xnote_add

    Nuxeo Platform will now use Elasticsearch 2.x versions and ability to use the REST inteface of Elasticsearch to improve facilities on security management for ops. Elasticsearch 2.X provides more features around data aggregation that can be used through our Elasticsearch API passthrough.

    Query

  • Annotations on Document Viewersnote_add

    Annotation support is added to the new set of documents viewers that have been added recently to the platform. Annotations covers geometrical annotations, textual annotations, etc...

  • Image Previewernote_add

    An Image Previewer is made available for allowing each time there is an image displayed in Nuxeo Web UI to:
    - zoom in
    - zoom out
    - rotate
    This Previewer would be used everywhere a picture can be downloaded (mostly the list of renditions).
    trending_upMore details on the specifications.

    Web UI

  • Tree Versionningnote_add

    It is possible to version a complete hierarchy of document. This is typically useful in configuration management type of project, as well as when using Nuxeo Platform to manage live content of a website, so as to be able to snapshot in a performant and robust manner the state of the whole site in a single api operation.

    Repository

  • Activity Streamsnote_add

    Activity Streams are intended to be a user friendly history track of what happens in the repository. Activities are quite similar to audit log entries, with the goal of being less verbose, nicely formatted, with easily translatable sentences, and a native ability to be aggregated at a folder or workspace level. Custom Activities can be fired easily from an automation chain.

    Repository


  • Microsoft Office Server based PDF conversionnote_add

    Office files are transformed in PDF using Microsoft Office server rather than Libre Office to improve the quality of the rendering.
